# movie_recommender
Movie Recommendation System using Linear Algebra (SVD, Gram-Schmidt, Least Squares)
# 🎬 Movie Recommendation System

This project uses Linear Algebra concepts to recommend movies.

## Concepts Used
- Gaussian Elimination
- LU Decomposition
- Gram-Schmidt
- SVD
- Least Squares

## Dataset

This project uses the MovieLens 100K dataset.

Download it from:
https://grouplens.org/datasets/movielens/100k/

After downloading, extract and place the folder like this:

ml-100k/
├── u.data
├── u.item

Make sure the folder is in the same directory as the Python file.
